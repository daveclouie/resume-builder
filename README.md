# Resume Builder

An AI-powered resume, cover letter, and portfolio builder — built as part of a six-project software engineering roadmap focused on modern full-stack development and AI integration.

## Status

🚧 Actively in development.

## Tech Stack

- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Database:** PostgreSQL
- **ORM:** Prisma
- **Deployment:** Vercel
- **CI/CD:** GitHub Actions
- **Tooling:** ESLint, Prettier, Husky, lint-staged

## Features (planned)

- [ ] Resume Builder (form + live preview)
- [ ] PDF Export
- [ ] Cover Letter Generator
- [ ] LinkedIn Summary Generator
- [ ] Portfolio Builder
- [ ] Personal Website Generator
- [ ] Theme Editor
- [ ] AI Content Suggestions
- [ ] Version History

## Getting Started

\`\`\`bash
npm install
npm run dev
\`\`\`

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Development

This project uses:

- **Prettier** for code formatting
- **ESLint** for linting
- **Husky + lint-staged** to auto-format and lint on every commit
- **GitHub Actions** to verify builds on every push
